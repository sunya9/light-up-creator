# Light Up Creator

Light Up (美術館 / Akari) パズルの作成・プレイ・共有ができるブラウザツールです。

## Features

- Editor — パズルの作成、唯一解の検証
- Player — URL を共有してブラウザ上でパズルをプレイ
- Puzzle Stock — localStorage にパズルを保存・管理

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, shadcn/ui, Konva

## Development

```bash
pnpm install
pnpm dev
```

```bash
pnpm test        # run tests
pnpm typecheck   # type check
pnpm lint        # lint
pnpm build       # production build
```
